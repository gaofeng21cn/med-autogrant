# Grant Authoring Skill Declaration

Skill owner: Med Auto Grant.

The domain skill declares MAG's grant-authoring capability boundary. It is not the professional-method source for each stage.

Professional grant judgment lives in repo-local Codex skills under `agent/professional_skills/`:

- `mag-call-fit-analyst`
- `mag-fundability-strategist`
- `mag-specific-aims-architect`
- `mag-proposal-section-author`
- `mag-grant-reviewer`
- `mag-rebuttal-planner`
- `mag-submission-package-auditor`

The domain skill covers specified-call intake, fundability strategy, specific aims, proposal authoring, reviewer-style critique, rebuttal planning, and submission-ready package preparation. Stage prompts route work and define accepted refs/blockers; professional skills provide specialist grant methods; tools only provide bounded affordances.

Generated OPL skill surfaces may expose descriptors and commands that call MAG handlers. They cannot become grant truth owner, quality verdict owner, export verdict owner, package authority, memory accept/reject authority, or owner receipt signer.
