# Authority Boundaries

MAG owns grant truth, fundability verdicts, authoring quality verdicts, export verdicts, package authority, grant strategy memory body and accept/reject decisions, owner receipts, transition oracle semantics, and grant-native helpers.

OPL owns generated surfaces, provider runtime, queue, attempt ledger, generic transition runner, workspace/source shell, memory locator transport, artifact/package lifecycle shell, operator projection, observability, and App/workbench shell.

The allowed MAG programmatic role is to validate refs, materialize refs, sign receipts, return typed blockers, and call grant-native helpers. Programmatic ready verdict generation is forbidden.
