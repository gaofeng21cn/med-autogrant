# Export And Package Gate

Owner: Med Auto Grant.

Submission-ready export is a package-stage authority decision. The local package command may materialize grant-owned refs after the MAG gate passes, and may return typed blockers when required sections, evidence, or portal-facing materials are missing.

OPL generated surfaces and artifact lifecycle shells consume refs and receipts. They cannot declare export readiness from package existence, provider completion, or generic lifecycle completion.
