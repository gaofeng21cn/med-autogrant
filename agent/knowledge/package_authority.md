# Package Authority Knowledge

MAG package authority covers local submission-ready export, grant-owned package refs, manual portal boundary, required blocker shapes, and owner receipt requirements.

Mechanical package completeness is only a lower-bound package check. MAG submission-ready output still requires owner-backed `submission_ready_export_verdict` provenance.

The package layer separates scientific review-ready completion from stricter local submission-ready export. External funding portal submission remains human-supervised and outside MAG's automated completion claim.
